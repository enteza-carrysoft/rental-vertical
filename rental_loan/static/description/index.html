Summary

Extension of module action_loan for rental use cases

Description

This module adds a smartbutton to the product form view to get an overview of all loans (from module account_loan in repository account_financial_tools) that are linked to this specific product.

Usage

This module is automatically installed when all of the following modules are installed in a database:

  • account_loan
  • rental_base
